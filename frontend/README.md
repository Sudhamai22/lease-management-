# Frontend (static)

This is a minimal static frontend for the Lease Management project.

How to use:

- Serve the `frontend` directory by a static server (for example `python -m http.server 3000`),
- Open `http://localhost:3000/index.html`, and click the "Fetch /api/health" button.
- Ensure the backend is running (e.g., Spring Boot) and CORS is configured if serving from a different origin.
